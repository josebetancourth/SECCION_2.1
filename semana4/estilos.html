Tipos de selectores

basicos

Dependiendo del tipo de selector puedo modificar la apariencia de varias maneras por ejemplo usando: Id, Class o etiqueta

ID

Para modificar elementos visuales usando su id iniciamos con el caracter # para mas informacion ver: ENLACE

CLASS

Para modificar elementos visuales usando su clase iniciamos con el caracter . para mas informacion ver: ENLACE

ETIQUETA

Para modificar elementos visuales usando su etiqueta iniciamos con el nombre de la etiqueta div, p, h1, body, etc para mas informacion ver: ENLACE
